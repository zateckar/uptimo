Configuration Help
Email Setup

Configure email notifications using SendGrid:

  • Simple: Just provide recipient email address
  • Reliable: SendGrid handles delivery and bounce management
  • Setup: Configure SENDGRID_API_KEY in environment
  • Optional: Set custom from email for branding
Telegram Setup

Get your Telegram bot configured:

  1. Message @BotFather on Telegram
  2. Create a bot and get its token
  3. Get your chat ID by messaging your bot
  4. Use @userinfobot to find your chat ID
Slack Setup

Configure Slack incoming webhooks:

  1. Go to Slack App Directory
  2. Create an "Incoming Webhooks" app
  3. Add it to your workspace
  4. Copy the webhook URL provided
